WASD - movement
Z - Free mouse from camera
X - Lock mouse to camera
C - Toggle ScaleHuman model
L - Delete cache (needed if u made changes to a model without using a new name for it)
+ / - - Change camera speed
Arrow keys - Change direction of sun-light
1 - Enable light on camera
T - increase camera-light intensity
Y - decrease camera-light intensity
B - increase ambient light
N - decrease ambient light
